狐狸的窗户
(日)安房直子 文,(日)织茂恭子 绘 / 2009-8 二十一世纪出版社 / 9.5 分

小时候看《儿童文学》，每次有刊载安房直子的短篇童话故事都欢呼鼓舞，读起来像舔舐棉花糖一样小心翼翼，暖与香的气息扑了满脸。长大以后再看安房直子就有一种成年人式的 “打劫大超市” 的快乐，不用再害怕太短太快读完，可以扑进整座棉花糖的房子乐不思蜀。

叶甫盖尼 · 奥涅金
[俄] 普希金 / 1985-3 人民文学出版社 / 8.6 分

叶甫盖尼 · 奥涅金是普希金的一部诗体小说，讲述 “多余人” 形象叶甫盖尼 · 奥涅金的故事。这一次是因为五月要去上海文广看奥涅金的话剧，所以紧急对原著进行补课，防止到时会有理解上的问题。

诗体小说真是非常奇妙的体裁，好像中世纪的游吟诗人坐在广场的喷泉前唱歌，最后叶甫盖尼又见到了达其雅娜，场面如同烈火一样极其绚烂璀璨，抓人极了。

窗边的小豆豆
[日] 黑柳彻子 著, 岩崎千弘 图 / 2003-1 南海出版公司 / 8.7 分

除夕的时候在街上值班，手冻得通红通红，转移注意力就抓了一本好读的书来重温。小豆豆的故事让人看到对孩子的无尽耐心，理解和爱，无论在什么年龄段去读都暖洋洋的。

做了一年老师以后来读倒有了很新鲜的感受，大部分都是“现在的学校敢这么做一定会被家长投诉……”

50：伟大的短篇小说们
欧 · 亨利, 契诃夫, 都德, 蒲宁, 舒尔茨, 波德莱尔, 莫泊桑, 马克 · 吐温, 泰戈尔, 海明威, 福克纳, 普希金, 托尔斯泰, 陀思妥耶夫斯基, 爱伦 · 坡, 伍尔夫, 霍桑, 毛姆, 卡夫卡, 茨威格, 狄更斯, 芥川龙之介, 菲茨杰拉德, 洛夫克拉夫特 / 2017-12 天津人民出版社 / 8.0 分

读短篇小说集，尤其是囊括了许多作家的短篇小说集，就像在超市享受试吃一样。东尝一口西尝一口，读来读去尝来尝去，挑挑选选间有大快乐。

只是有时候也会吃到很难接受的好味，比如我好难接受卡夫卡哦……

2001：太空漫游
(英)阿瑟 · 克拉克 / 2014-5-1 上海人民出版社 / 9.3 分

看《流浪地球》，想起来顺便把这一本看了。所谓的太空浪漫被它写到了极致，真真正正“危楼高百尺，手可摘星辰，不敢高声语，恐惊天上人。“。

宇宙到深处就近乎哲学，而哲学到深处又何尝不是宇宙本身呢。我可太喜欢这本书了，我要给它一个大大的五星。

来自新世界 （上下）
[日] 贵志祐介 / 2014-4 上海译文出版社 / 9.1 分

捕捉到拟蓑白的时候觉得实在转折得漂亮，顷刻间让充满民俗风味的风情小说染上了科幻色彩，百分之五十的时候对异化生物的描述也很漂亮，好久没有这种又惧又怕却撑着不愿意睡地通宵看书的感觉了。

后半部对比上半部来说就乏力许多，可能我对科幻作品的期待还是在于想象力，当它转折成一个冒险救世的故事的时候，对我来说就完全没有意思了，好失望哦。

心
[日]夏目漱石 / 2013-8-1 陕西师范大学出版总社 / 8.7 分

无论在哪一本书中，我都超级喜欢以 “先生” 形象出现的角色！也非常喜欢《心》中先生刚出场的时候的描写，这样的先生刚出场时却伴着一个西洋人，种种内在激荡的感情与微妙的情绪非常能够打动人。

在书中不断面对死亡的感觉仿佛佛寺当中不断叩问本心的钟声，读完颇有一种死里逃生的释然感。我还没有准备好去思考生与死的关系，也无法去探讨如何面对死亡。再过几年读《心》或许会有不同的感受，但是现在我最好还是就停在这里好啦。

球状闪电
刘慈欣 / 2005-6 四川科学技术出版社 / 8.7 分

朋友提起《球状闪电》中，刘慈欣难得塑造了一个正面丰富的女性形象，于是做头发正好有时间，就翻开来看。我总觉得刘慈欣非常擅长描写很纯粹的东西，比如在这本书中，理想主义者的光辉简直压倒性地让人觉得无法承受，每每需要停下来喘一口气。

到了最后，瓶中的一朵量子玫瑰犹带露水，这也是科幻世界中的一种极致浪漫了
